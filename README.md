# QuickBooks App Site

Static pages for Intuit production app setup.

## Files
- `index.html`
- `privacy.html`
- `terms.html`
- `connect.html`
- `disconnect.html`

## Before publishing
Replace `YOUR_EMAIL` in:
- `privacy.html`
- `terms.html`

## Suggested GitHub Pages repo name
- `quickbooks-app-site`

## Intuit form mapping
If your GitHub Pages site is:
- `https://YOURNAME.github.io/quickbooks-app-site/`

Use:
- Host domain: `YOURNAME.github.io`
- Launch URL: `https://YOURNAME.github.io/quickbooks-app-site/`
- Disconnect URL: `https://YOURNAME.github.io/quickbooks-app-site/disconnect.html`
- Connect/Reconnect URL: `https://YOURNAME.github.io/quickbooks-app-site/connect.html`
- Privacy Policy: `https://YOURNAME.github.io/quickbooks-app-site/privacy.html`
- Terms / EULA: `https://YOURNAME.github.io/quickbooks-app-site/terms.html`
